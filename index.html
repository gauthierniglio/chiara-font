02 — Install

One file.
Every weight. 
@font-face {
  font-family: 'Chiara';
  src: url('cdn.jsdelivr.net/gh/gauthierniglio/
       chiara-font@v1.0/web/ChiaraVF.woff2')
       format('woff2');
  font-weight: 100 900;
  font-display: swap;
}
<!-- Download from GitHub -->
<link rel="stylesheet"
      href="chiara.css">

<!-- Or via jsDelivr CDN -->
<style>
  @import url('https://cdn.jsdelivr.net/
  gh/gauthierniglio/chiara-font
  @v1.0/web/chiara.css');
</style>
/* Standard weight */
h1 {
  font-family: 'Chiara', sans-serif;
  font-weight: 700;
}

/* Variable axis — fine control */
.display {
  font-variation-settings: 'wght' 850;
}
